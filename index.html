Selamat Datang di WebGIS Admin

Halaman admin untuk mengelola data rumah tangga, tempat ibadah, dan memverifikasi laporan dari masyarakat.

Menambah Tempat Ibadah

Klik tab "Ibadah" di sidebar, lalu klik lokasi pada peta. Form akan muncul untuk mengisi detail.

Menambah Rumah Tangga

Klik tab "Rumah", lalu klik lokasi pada peta. Skor kemiskinan dihitung otomatis berdasarkan data yang diisi.

Verifikasi Laporan Publik

Klik Panel Admin → Laporan Publik untuk melihat laporan dari masyarakat. Setujui untuk langsung menambah data ke peta.

Dashboard & Laporan

Klik tab "Chart" untuk grafik analitik. Marker dapat diseret untuk memindahkan posisi. Radius dapat diubah langsung dari popup.
